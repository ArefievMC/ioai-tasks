# IOAI Task Bank

Static catalogue of AI olympiad / Kaggle competition tasks (171 entries) with
topic tags from the IOAI syllabus.

Deploy: any static-site host (Render, Cloudflare Pages, GitHub Pages).
Open `index.html` — it loads `tasks_data.json` and renders a searchable grid.

Source data: `tasks_final.csv` (in upstream prep folder).
